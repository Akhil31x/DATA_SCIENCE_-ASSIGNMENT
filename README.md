# Seasonal Sale Time Series Analysis

## Project Overview
This project analyzes Amazon product sales trends using time series methods.  
It studies seasonal patterns, discount trends, and forecasts future sales using Holt Winters Exponential Smoothing.

## Domain
E Commerce Data Analytics

## Dataset
Amazon India Product Sales Dataset

## Objectives
- Analyze monthly sales trends
- Identify seasonal patterns
- Study discount behavior
- Decompose sales into trend, seasonal, and residual parts
- Forecast future sales

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Statsmodels

## How to Run

Step 1  
Install dependencies  
pip install -r requirements.txt  

Step 2  
Add amazon.csv inside data folder  

Step 3  
Run the script  
python src/seasonal_sales.py  

## Expected Output
Four plots saved in outputs folder  
Future monthly sales forecast displayed in terminal
